1.2. Node Paket Yöneticisi (npm)

Node Package Manager (NPM) iki ana özellik sağlar:

  • www.npmjs.com adresi üzerinden Node.js paketlerine / modüllerine ulaşma
  • Node.js paketlerini yüklemek için komut satırı yardımcı programı, sürüm yönetimi ve Node.js paketlerinin bağımlılık yönetimi

Node.js kurulum paketleri ile birlikte gelir. Hangi kurulum paket versiyonunun olduğunu öğrenmek için aşağıdaki kodu “PowerShell” yönetici  modunda açın. Tabii ki bulunduğunuz dizi önemlidir. (cd ile “yol” yazıp klasöre geçiş yapın.) Ayrıca, Set-ExecutionPolicy Unrestricted -Scope CurrentUser -Force yazıp enter tuşuna basın.
